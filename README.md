mikutter-santa
============

これなに
--------

![miku!](https://cloud.githubusercontent.com/assets/4361134/11957368/c18a511e-a904-11e5-907f-b104657f9204.png)

クリスマスが近くなるとみくったーちゃんがサンタ帽をかぶります。

インストール
-----------

```sh
cd ~/.mikutter/plugin/
git clone https://github.com/pocke/mikutter-santa
```

mikutter を再起動してください
